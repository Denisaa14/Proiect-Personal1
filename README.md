Proiectul prezinta urmatoarele functionalitati:
-Autentificare și Securitate: Sistem de Login/Register integrat folosind ASP.NET Core Identity.

-Izolarea Datelor (Multi-tenancy): Fiecare utilizator are acces exclusiv la propriile date, asigurat prin filtrare la nivel de bază de date (UserId).

-Dashboard Interactiv: Vizualizarea cheltuielilor pe categorii folosind grafice dinamice (Chart.js/Canvas).

-Sistem CRUD Complet: Adăugarea, editarea, ștergerea și căutarea tranzacțiilor financiare.

-Analiză Financiară: Calcularea automată a veniturilor totale, a cheltuielilor și a balanței curente.
